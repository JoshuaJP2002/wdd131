Elite Auto Dealer

Website Planning Document

Site Name

Elite Auto Dealer

I chose this name because the website represents a luxury car dealership that sells high-performance super cars. The word "Elite" shows exclusivity and high quality vehicles, and "Auto Dealer" clearly explains the business purpose.

Optional domain: eliteautodealer.com

Site Purpose

The website will represent a luxury supercar dealership. Visitors will be able to browse available vehicles, view specifications, see prices, and save cars to a wishlist. The website will also include a contact form that allows users to request more information or schedule a test drive. The site will use JavaScript to display inventory dynamically and store selected vehicles using localStorage.

Scenarios

  • What supercars are currently available for sale?
  • How can I contact the dealership to schedule a test drive?
  • Can I save the cars I am interested in and view them later?

Color Schema

  • Black (#0f172a) main background and header
  • Red (#dc2626) buttons, links, and accents
  • White (#f8fafc) text color
  • Gray (#334155) card backgrounds

Typography

  • Oswald – headings and titles
  • Roboto – body text and paragraphs
